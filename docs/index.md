# OllamaCode

A coding assistant powered by **local models** (Ollama) and **MCP** (Model Context Protocol).

## Quick start

```bash
uv sync
uv run ollamacode "Explain recursion in one sentence"
```

See the project README for installation, usage, and options.

## Wiki

The [Wiki](WIKI.md) collects links to configuration, MCP, TUI, API, and advanced docs.

## API

The [API reference](api.md) documents the public Python API: agent loop, bridge, MCP client, and CLI.
